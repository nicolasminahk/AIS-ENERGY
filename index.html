Las empresas invierten en renovables
pero no pueden demostrar su impacto

El mercado de certificación climática es opaco, tiene granularidad anual y no valora el almacenamiento. Resultado: el BESS no tiene economía propia.

⏱

Sin trazabilidad horaria

Los certificados actuales (GOs) tienen granularidad anual. No pueden demostrar que la energía fue verde en cada hora específica del día.

🔍

CO₂ no verificable

Sin datos SCADA horarios validados por Red Eléctrica, el impacto en emisiones no puede auditarse ni certificarse con rigor técnico.

⚡

BESS sin economía propia

El almacenamiento depende de arbitraje de mercado. El valor medioambiental real de descargar en horas de alta emisión queda sin remunerar.

Factor de emisión España · Variación horaria (g CO₂/kWh) · Día tipo

La diferencia entre las 3h (80 g/kWh) y las 20h (400 g/kWh) es 5× mayor.
Solo un sistema con granularidad horaria puede capturar este valor diferencial.

La solución: BESS-CO₂ Token

1 BESS-CO₂ Token = 1 tonelada de CO₂ evitada. Medido con datos SCADA reales, calculado hora a hora, registrado en blockchain.

Fórmula de emisión certificada
Tokens = Edesc × Femisión × 0,95
Edesc = Energía descargada (kWh, datos SCADA horarios)  ·  Femisión = Factor CO₂ Red Eléctrica (g/kWh, API REE)  ·  0,95 = Eficiencia ciclo BESS
01

SCADA horario

Los medidores inteligentes registran energía cargada y descargada del BESS cada hora con precisión ±0,1%.

02

API Red Eléctrica

Se obtiene automáticamente el factor de emisión marginal de la red española (REE) para cada franja horaria.

03

Cálculo verificado

Python calcula los tokens según la fórmula certificada. El código es auditable y open-source para clientes.

04

Smart contract

Los tokens se acuñan en Polygon mediante contrato Solidity auditado. Trazabilidad inmutable on-chain.

05

Certificado digital

El propietario del BESS recibe un certificado PDF con hash blockchain verificable, exportable a ESG reporting.

06

Dashboard en tiempo real

Panel React con métricas live: tokens acumulados, impacto CO₂, valor de mercado y trazabilidad completa.

🔧 Arquitectura técnica del sistema (detalles)
Capa 1

Sensores & Hardware

Medidores inteligentes clase A, inversores con protocolo Modbus/RS-485, monitorización cada 15min.

Capa 2

SCADA & Adquisición

Sistema supervisorio Python. Normalización, validación y almacenamiento en base de datos time-series.

Capa 3

API Red Eléctrica

Conexión a ESIOS (REE) para factor de emisión marginal horario. Actualización automática cada hora.

Capa 4

Smart Contract

Contrato ERC-1155 en Polygon. Emisión, transferencia y quema de tokens auditada y verificable on-chain.

Capa 5

Dashboard & API

Frontend React con métricas live. API REST para integración con sistemas ERP/ESG de clientes corporates.

Capa 6

Certificados & Reporting

Generación automática de PDF auditables con QR de verificación blockchain. Exportación CSRD/GRI.

¿Para quién es BESS-CO₂ Token?

Diseñado para organizaciones con objetivos Net Zero que ya tienen o planean instalaciones BESS y necesitan demostrar impacto climático verificable.

Prioridad #1

Data Centers & Cloud

Los data centers son los mayores consumidores de energía y los más presionados por regulación e inversores para demostrar descarbonización real. El BESS es infraestructura crítica que ya tienen. Solo les falta la capa de certificación horaria.

✓

Cumplimiento CSRD con evidencia on-chain auditada, no estimaciones

✓

Diferenciación competitiva real: "Power Usage Effectiveness + CO₂ Avoidance"

✓

Ingresos adicionales por venta de tokens a corporates en su cadena de valor

✓

Integración API con dashboards ESG existentes (Salesforce Net Zero, SAP GreenLedger)

🏭

Industria intensiva

Cemento, acero, química, papel. Alta exposición a EU ETS y mecanismo CBAM. El BESS-CO₂ Token complementa la estrategia de reducción de huella con activos ya instalados.

🏢

Corporates Net Zero

Grandes empresas con compromisos SBTi o Net Zero 2040–2050. Necesitan credenciales climáticas verificables para clientes, reguladores e inversores ESG.

⚡

Utilities & ESCOs

Operadores de parques de almacenamiento a gran escala. Modelo B2B2C: certifican en bloque y revenden tokens a sus clientes industriales y comerciales.
